<h1>Command Guide </h1>

```vim
:OllieChat
```

It Opens the chat window anywhere, in bigger screen, with rounded border.
<br> It also include chat with model switching feature.

<br>

```vim
:OllieModel
```

It opens the model-switching window, allowing users to see available model.

<br>

```vim
:OlliePanel
```

It display the panel on left/right of the full window. Allowing users to use panel and edit the file at same time.

<br>

```
:OllieHealth
```

It runs a test on network, system information and manage to give good recommandation for your best model.

<br>

```
:OllieHistory
```

It displays chat history and information of past conversation with date and time.
